## Dependencies

-   [Rustup](https://rustup.rs/)
-   Tauri Cli `cargo install tauri-cli`

## Developing

With local development server

```bash
cargo tauri dev
```

Without local development server

```bash
cargo run
```

## Building for Release

```bash
cargo tauri build
```
